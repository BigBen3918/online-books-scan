# Google Books Search App

You can search any books with this app.

## Usage

Clone the project

```bash
  git clone https://github.com/BigBen3918/online-books-scan.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
  yarn install
```

Start the server

```bash
  npm run start
  yarn start
```
